# Stone-Paper-Scissors-Game
✊✋✌ Stone Paper Scissors Game  A simple and interactive Stone Paper Scissors game built using HTML, CSS, and JavaScript. The user can play against the computer, and the result is displayed instantly along with score updates.

🎮 Features

Play Stone, Paper, Scissors against the computer

Random computer choice generation
Instant result display (Win / Lose / Draw)
Score tracking system
Simple and clean user interface
Fully runs in the browser

🛠 Tech Stack

HTML – Structure of the game

CSS – Styling and layout

JavaScript – Game logic and interactivity

⚙ How the Game Works
The player selects Stone, Paper, or Scissors.
The computer randomly generates its choice.
The program compares both choices using game rules:
Stone beats Scissors
Scissors beats Paper
Paper beats Stone
The result (Win / Lose / Draw) is displayed and the score updates.
